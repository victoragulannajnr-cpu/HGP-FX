## 📌 Project Overview

HGP FX is a modern web-based financial terminal built with HTML, TailwindCSS, and Firebase.  
It simulates a structured investment allocation system with authentication, real-time database syncing, and dynamic portfolio calculations.

The platform includes:

- Secure user authentication (Email/Password + Google Sign-In)
- Real-time Firestore database integration
- Investment plan simulation engine
- Portfolio tracking dashboard
- Deposit & withdrawal UI workflow (simulated ledger system)
